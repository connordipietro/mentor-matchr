# Mentor Matcher

This tool is designed to encourage software membership.

## Concepts Introduced

Stripe, Sockets.IO, Material-UI, Image Upload, Drag and Drop, Admin Dashboard

## Tech
This is a continuously deployed MERN app (Heroku, MongoDB Cloud Atlas).
